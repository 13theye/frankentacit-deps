13thlib is a library for 13th Eye creative coding projects.

Frankentacit-deps is a snapsot of 13thlib frozen for the FrankenTacit suite of performances. Although fully functional in that context, it is not complete and not intended for use in new projects.

Components:
- Nnpipe: a Nannou-based drawing, rending, and post-processing pipeline
- Prat: a stable and accurate Ableton-Link compatible clock
- FPS: a frames-per-sec counting module
